# React E-Commerce Store [ Frontend ]

## Technology used:
* Next.js
* React.js
* Stripe
* Auth0
* Framer Motion
* Styled Components

## To run
* Install dependencies
* Npm run dev
* Check localhost ( http://localhost:3000/ )
